# 🔒 Ation Gerador de Senhas

Aplicação desenvolvida em NextJS para geração de senhas seguras com possibilidade de customização conforme necessidade do usuário.

Por padrão a aplicação gera senhas numéricas, porém o usuário pode incluir as seguintes variações:

- Quantidade de caracteres;
- Letras Maiúsculas;
- Letras Minúsculas;
- Símbolos.

## 📦 Rodando o projeto

Faça um fork deste projeto, clone o repositório para sua máquina, entre na pasta do projeto `cd passwordgen`  instale as dependências `npm install` e rode a aplicação em modo de desenvolvedor `npm run dev` para realizar as customizações que entender necessário.
